Course Curriculum

Welcome to the Web Development course!

This curriculum is designed to help you master the fundamentals of HTML, CSS, and JavaScript. You'll learn how to build interactive and responsive websites from scratch.

Course Objectives

  • Understand HTML structure & semantics
  • Apply CSS for styling and layout
  • Use JavaScript for dynamic behavior

Modules Overview

  1. Introduction to HTML
  2. Styling with CSS
  3. JavaScript Basics
  4. DOM Manipulation
  5. Project Development

Key Terminology

HTML
HyperText Markup Language used to structure web content.
CSS
Cascading Style Sheets used to style HTML elements.
JavaScript
Programming language used to add interactivity to web pages.
